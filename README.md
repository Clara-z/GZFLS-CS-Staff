# GZFLS-CS-Staff

1. 每个人开一个自己的branch，写完自己的part以后提交 Pull Request 与主分支进行合并

2. Pull Request 做 comment 的时候一定要写上自己负责的part，这样与别人版本冲突的时候才方便处理冲突。

3. 如果你的part中有**代码文件** / **代码片段**， 一定要用 \`\` 围起来
    
    例子1:
    
    \`function()\` ---> `function()`
    
    例子2: 
    
    \`\`\`java
    
    public static void main(String[] args){
    
      //...
      
    }
    
    \`\`\`
            
     ```java
     public static void main(String[] args){
        //...
     }
     ```
